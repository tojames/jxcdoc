# 成本管理
### 销售退货单指定成本
因为调账等特殊情况，用户可对销售退货单中的存货进行手工指定成本。

操作节点：业务处理－成本管理－销售退货单指定成本

1、销售退货单的存货通过手工指定成本后，将不会根据成本计算的结果变更

### 成本计算

在进销存业务月末结转以前，需要将企业存货按存货核算方式计算其出入库成本，然后再进行期末结转处理。

操作节点：业务处理－成本计算

1、进入操作节点，选择需要计算成本的存货

2、点击【下一步】按钮，系统自动开始计算所选择存货的单位成本，最后【完成】成本计算 。
