# 实物仓库
### 仓库入库单
用于处理企业实物仓库存货数量入库。

操作：

业务处理—实物仓库—仓库入库单

操作说明：

1、该单据可以选单生成，也可以直接进行录入；

2、单据过账后可以通过菜单栏上的“生成单据”生成相关单据。例如：“仓库入库单”可以生成进货单、其他入库单等。
